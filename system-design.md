## HLD Flow

### Poll Creator

- Logs in via OIDC
- Creates poll
- Adds questions/options
- Shares public link
- Watches analytics
- Publishes results

---

### Public User

- Opens shared poll link
- Submits response
- Sees final results after publish

---

### Backend

Handles:

- auth validation
- poll CRUD
- response validation
- analytics
- publish result

---

### Socket.IO

Handles:

- live response count
- analytics updates
